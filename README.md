# VetSoft 🐾

Sistema de gestión veterinaria desarrollado como proyecto de práctica para aplicar los conceptos aprendidos en la asignatura Seminario de Lenguajes II.

## Características

- Gestión de propietarios
- Gestión de mascotas
- Relación Propietario ↔ Mascota
- API REST para operaciones CRUD
- Persistencia en base de datos relacional
- Configuración del negocio mediante archivos JSON
- Interfaz de usuario desarrollada con Blazor WebAssembly

## Tecnologías

- C#
- ASP.NET Core
- Entity Framework Core
- Blazor WebAssembly
- PostgreSQL
- Docker
- Swagger

## Arquitectura

El proyecto se encuentra organizado en capas:

- API (Endpoints)
- Lógica de negocio
- Persistencia de datos

## Objetivo

El propósito del proyecto es simular la gestión básica de una veterinaria, permitiendo administrar propietarios y mascotas mediante una arquitectura desacoplada y buenas prácticas de desarrollo.
